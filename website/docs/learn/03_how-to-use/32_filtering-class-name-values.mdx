# Filtering class name selectors

Example: Remove all but `border*` selectors and add `font-bold` selector:

<CodeEditor
  live
  resultClassName={`{(state, provided) => [
  ...provided.filter(
    selector => !selector.startsWith('border')
  ),
  'font-bold',
]}`}
>{_Add}</CodeEditor>
